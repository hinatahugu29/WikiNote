## 🚀 アプリの概要

このパーソナルWikiは、ブラウザひとつで動作する高機能なメモアプリです。

- サーバーやデータベースは不要
- HTMLファイルを開くだけで、すぐに使い始められます
- Wikiスタイルのリンク機能、グラフビュー、Markdown記法など、知識を整理・活用するための機能が凝縮されています

## 📖 基本操作ガイド

### キーボードショートカット

| キー | 機能 |
|------|------|
| `Ctrl` + `S` | 記事の保存 |
| `Esc` | モーダルを閉じる / プレビューから戻る |
| `Ctrl` + `V` | 画像の貼り付け |

### フォルダ（カテゴリ）の使い方

編集画面の「フォルダ」欄にスラッシュ区切りでパスを入力すると、自動的に階層構造が作られます。

例：`プロジェクトA/議事録/2026`

**📁 フォルダ挿入ボタン**：サイドバーのツリー内の各フォルダ名の横にある 📁 ボタンをクリックすると、そのフォルダのパスが編集中の記事の「フォルダ」欄に自動的に入力されます。手入力不要で、素早く整理できます。

### Wikiリンクの挿入

| 方法 | 説明 |
|------|------|
| **ツールバー「🔗」ボタン** | クリックで `[[]]` を挿入。カーソルが中央に配置され、すぐにリンク先を入力できます |
| **サイドバー「🔗」ボタン** | 記事タイトル横のボタンをクリックで `[[記事タイトル]]` を直接挿入 |

### Markdown記法

| 記法 | 書き方 |
|------|--------|
| **見出し** | `# H1`, `## H2` |
| **太字** | `**Bold**` |
| **リスト** | `- Item` |
| **Wikiリンク** | `[[記事タイトル]]` |
| **外部リンク** | `[表示名](URL)` |
| **数式** | `$$ E=mc^2 $$` |
| **コード** | ` ```js ... ``` ` |

### データの管理

データはブラウザの `localStorage` に保存されます（最大約5MB）。

- **画面上部に現在の使用量が表示されます**（50%超で黄色、80%超で赤色の警告表示）
- **定期的なバックアップを推奨します**

| 機能 | 説明 |
|------|------|
| **バックアップ保存** | 画面上部の「💾 バックアップ保存」をクリック。タイムスタンプ付きのJSONファイルが即座にダウンロードされます |
| **復元** | 「📂 復元」からJSONファイルを読み込んでデータを復元 |
| **バックアップ促進** | 最後のバックアップから7日以上経過すると、起動時にリマインダーが表示されます |

## 💡 もっと使いこなすTips

### ⚡ クイックナビゲーション

編集画面・プレビュー画面の左上にある `☰` ボタンを押すと、サイドバーが開閉します。
いちいち一覧画面に戻らなくても、ツリーから別の記事を選んですぐに切り替えることができます。

**便利な挙動:**
- **編集モードの維持**: 記事を編集中にサイドバーから別の記事を選ぶと、編集モードのまま切り替わります。連続して記事を修正したい時に便利です。
- **サイドバーのリサイズ**: サイドバーの右端をドラッグすると幅を自由に変更できます（モバイル版を除く）。幅の設定は記憶されます。

### ➕ 編集中からの新規作成

編集画面のフッターにある「**＋ 新規**」ボタンをクリックすると、一覧に戻らずに直接新しい記事の作成を開始できます。
思いついたらすぐに新規記事を作成できるので、アイデアを逃しません。

### 🏷️ 入力欄のラベル表示

「フォルダ」「タイトル」「タグ」の各入力欄の右端には、薄いラベルが常に表示されています。
文字を入力した後でも「この欄は何の欄だったか？」と迷わずに済みます。

### 🖥️ 没入型エディタ

エディタ画面の右上にある `⛶` ボタンを押すと、画面いっぱいに編集エリアを広げることができます。
また、「エディタ」タブを選ぶだけでも、自動的に全幅モードになり、執筆に集中できます。

### 🔄 スクロール同期

エディタ画面とプレビュー画面のスクロール位置が連動します。
片方をスクロールすると、もう片方も対応する位置まで自動的に移動するため、編集中の箇所がプレビューでどう見えるかを常に確認できます。
- ツールバーの `🔗` アイコンをクリックすることで、同期のON/OFFを切り替えられます。

### 🔴 赤リンクを活用しよう

記事の中に `[[未来のアイデア]]` のように、まだ存在しないページのリンクを書いておくと、リンクが赤く表示されます。
これは「あとで書くことリスト」として機能し、クリックするだけでそのタイトルの新規作成画面が開きます。

### 🔎 検索のコツ

- 検索ボックスでは、スペース区切りで複数の単語を入力すると「AND検索」になります
- **サイドバーの検索は本文も対象**になります。タイトルに含まれないキーワードでも記事を見つけられます
- 検索中、サイドバーのツリー表示は一時的に通常のリスト表示に切り替わり、マッチした記事だけが表示されます

### 🕸️ グラフビューを楽しむ

ホーム画面の「🕸️ グラフビュー」をクリックすると、記事同士のつながりをネットワーク図で可視化できます。
右上の「⚙️（歯車）」アイコンをクリックすると、物理演算やアニメーションの詳細な設定が可能です。

**没入モード**:
- 右上の操作ボタン（⚙️設定、✖閉じる）は、マウス操作がないと数秒後に自動的にフェードアウトして消えます。
- 再びマウスを右上に近づけると表示されます。これにより、グラフの鑑賞により没頭できます。

#### 📊 パラメータ解説

| 設定項目 | 効果 | 調整のヒント |
|---|---|---|
| **ノード間引力** | 反発し合う力 | マイナスを大きく（例: -5000）すると、銀河のように広大に広がります |
| **中心引力** | 中央に集まる力 | 1.0でギュッと凝縮。0.0にすると画面外へ果てしなく広がっていきます |
| **振動幅** | 漂う動きの大きさ | 大きくすると、生き物のように活発に動き回ります。0にすると静止します |
| **カメラ追従** | 視点の移動 | **ON**: 没入感のあるツアー。<br>**OFF**: 俯瞰して楽しむ「鑑賞モード」（酔いにくい） |

#### 🧪 おすすめ設定レシピ

- **🌌 銀河モード**
  - 広大な宇宙を感じたい時に
  - ノード間引力: `-4000` / 中心引力: `0.1` / カメラ追従: `OFF`

- **🧘 禅モード**
  - ゆっくりと漂う記事を眺めてリラックス
  - アニメ速度: `8000ms` / 振動幅: `50` / カメラ追従: `OFF`

- **🎢 ジェットコースター**
  - 知識の海を高速で飛び回る
  - アニメ速度: `1000ms` / ツアー間隔: `1500ms` / カメラ追従: `ON`
